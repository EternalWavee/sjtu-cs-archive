## Organization

To keep things straightforward, all materials are organized strictly by **Course Code** rather than by semester, year, or course name. 

## Content Variance

Please note that the content structure varies significantly from course to course based on what was required and how I studied:
- **Some folders** might only contain lab assignments, source code, or project files.
- **Other folders** might include comprehensive handwritten notes (exported as PDFs), summaries, and exam cheat sheets.
- There is no unified structure across different courses.

## Usage

Feel free to reference the code, read the notes, or use the materials to help with your own studies. If you spot any mistakes in the notes or bugs in the code, you are welcome to open an issue.
